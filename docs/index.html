Kestrel Mail

A KDE-native email client with modern UX, reliable IMAP sync, Google Calendar integration, and OAuth2-first account setup.

View on GitHub Build Instructions

Features

Full IMAP Sync

CONDSTORE incremental sync, IDLE for real-time updates, and per-account connection pooling for fast, reliable mail.

Google Calendar

Week view with live event cards, invite RSVP (accept, decline, tentative), and per-calendar visibility.

Drag and Drop

Move messages between folders, apply Gmail labels, or archive to local folders with drag and drop.

OAuth2 + KWallet

Secure OAuth2 with PKCE for Gmail. Tokens stored in KWallet, libsecret, or encrypted file fallback.

Local Folders

Archive messages locally with full body and attachment download. Survives server-side deletion.

KDE Native

Built with Qt 6, QML, and Kirigami. Follows KDE conventions with desktop notifications and system tray.
